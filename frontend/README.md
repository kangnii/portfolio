# Frontend (React)

Portfolio React (Vite) avec effets dynamiques (Framer Motion), fond animé (Canvas), et data chargée depuis l’API Flask.

## Démarrage (dev)

```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```

Vite : `http://localhost:5173`

## Variables

- `VITE_API_BASE=http://localhost:5000`
